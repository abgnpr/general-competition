# Descriptive Writing

Two-track system for descriptive/subjective sections across banking and insurance exams.

- **Track A — PO letter + essay**: SBI PO Mains, IBPS PO Mains, LIC AAO Mains.
- **Track B — Phase 2 theme-based writing**: RBI Grade B, NABARD, SEBI.

See the starter plan → [plans/descriptive-starter-plan.md](../../plans/descriptive-starter-plan.md)

## Structure

| Folder | Purpose |
|--------|---------|
| `track-a-po/` | Templates + reusable paragraphs for PO letter/essay |
| `track-b-phase2/` | Answer-writing structure for Phase 2 descriptive |
| `themes/` | One-page theme notes — the content base for Track B |
| `answer-log.md` | Log of written answers with feedback |
