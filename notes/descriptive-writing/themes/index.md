# Themes

One-page theme notes. The content base for Track B Phase 2 writing.

## Format for every theme

- **Definition / core concept** — 2 sentences.
- **Key data points** — 3–5 latest stats with year/source.
- **Schemes / committees / reports** — named references.
- **Current affairs linkage** — 2 recent items (last 6 months).
- **Counter-view / limitations**.
- **1-line conclusion hook**.
