# Track B — Phase 2 Theme-Based Writing

For RBI Grade B Phase 2, NABARD Phase 2, SEBI Phase 2.

## Answer structure (non-negotiable skeleton)

```text
[Introduction — 3-4 sentences]
- Define the concept / frame the issue.
- State why it's relevant now (recent policy / report / event).
- Preview what the answer will cover (1 sentence).

[Body — 3 to 5 sub-sections with sub-headings]
- Each sub-section: 1 idea, 1 data point, 1 example.
- Use sub-headings ("Monetary Transmission Channel", "Recent RBI Actions") — evaluators scan.
- Cover: dimensions of the issue, causes, implications, counter-view.

[Recommendations / Way Forward — 3-5 bullets]
- Specific, actionable. Avoid platitudes.
- Link to existing policy frameworks where possible.

[Conclusion — 2-3 sentences]
- Synthesize, don't repeat.
- End with a forward-looking statement.
```

## Word length by question type

| Marks | Expected words | Time |
|-------|---------------|------|
| 5 | 100–150 | 8 min |
| 10 | 250–300 | 15 min |
| 15 | 400–500 | 22 min |
| 30 | 800–1000 (essay) | 40 min |

## The "theme-first" principle

Don't prep questions. Prep themes. One good theme note answers 5–10 different questions.

Theme library → [themes/](../themes/)

## Evaluation criteria (what markers look for)

1. **Conceptual clarity** — do you actually understand monetary policy / NPAs / etc.?
2. **Data / examples** — at least one stat, one scheme/committee name per sub-section.
3. **Structure** — headings, logical flow, visible skeleton.
4. **Balance** — one counter-view or limitation shows maturity.
5. **Recent linkage** — current affairs reference (last 6 months).
6. **Recommendations** — specific, not vague.

## Do / Don't

**Do:**

- Use sub-headings. Always.
- Name schemes, committees, reports. *"The Nachiket Mor Committee recommended…"* beats *"A committee recommended…"*
- Cite at least one data point per 150 words.
- Keep sentences short under time pressure.

**Don't:**

- Write prose-only answers without headings (low scan-ability).
- Pad with jargon you don't understand.
- Forget the "Way Forward" section. Missing it loses easy marks.
- Copy UPSC-style essays. Banking Phase 2 is shorter, more structured.
