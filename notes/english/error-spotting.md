# Error Spotting / Sentence Improvement

**Tier 1 · Priority 4.** Direct application of Grammar Basics. Train the eye.

Status: not started

## Approach

1. Read the full sentence silently for "does it sound right".
2. Check the Big 5 grammar rules in order: S-V agreement → tense → preposition → article → pronoun.
3. If none obvious: check modifiers, parallelism, comparison.
4. Watch for redundancy (*"return back", "equally as", "more better"*).

## Scan Protocol

- **Subject & verb** — match them first, even across long phrases.
- **Tense markers** — "since", "yet", "already" (perfect); "yesterday", "ago" (simple past).
- **Preposition phrases** — test if the combination is standard.
- **Pronoun antecedent** — who/what is "it", "they", "he" referring to?
- **Articles before nouns** — is it specific or general?

## Common Traps

- Redundancy: *return back*, *repeat again*, *combine together*, *rise up*.
- Double negatives: *not... nothing* → should be *not... anything*.
- Incorrect comparative: *more better*, *most unique*, *as good or better than* (missing *as*).
- Unidiomatic: *discuss about*, *enter into a room*, *order for food* → remove the extra preposition.

## No-Error Questions

- Roughly 10–15% of Error Spotting has no error. Don't force an error if unsure; trust the scan.
