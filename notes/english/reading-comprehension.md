## Question-Type Library

| Type | Recognition cue | Strategy |
|------|----------------|----------|
| Main idea | "central theme", "best title", "primary purpose" | Read first + last para closely |
| Factual | "according to the passage", "which of the following is true" | Direct match; skim for keywords |
| Inference | "most likely", "implies", "suggests" | Read between lines; don't over-reach |
| Vocab-in-context | "meaning of word X as used in the passage" | Ignore dictionary meaning; use surrounding sentence |
| Tone | "author's attitude", "tone of the passage" | Watch for adjectives, loaded words |
| Structural | "why does the author mention X?", "the passage is organized as" | Think paragraph-level purpose |
