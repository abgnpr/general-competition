## Sub-types

- **Tabular** — dense numbers; approximation is king.
- **Bar graph** — single or grouped; eyeball ratios.
- **Line graph** — trends, differences year-on-year.
- **Pie chart** — degrees ↔ % (90°=25%, 60°=16.67%, 36°=10%).
- **Caselet** — data in paragraph; rewrite as a table FIRST.
- **Mixed DI** — combination; decide which sub-chart to use per question.
- **Missing DI** — some values hidden; solve sequentially.
