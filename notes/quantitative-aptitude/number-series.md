## The Pattern Library

| Pattern | Recognition cue | Example |
|---------|----------------|---------|
| Arithmetic (+constant) | Equal differences | 2, 5, 8, 11 |
| Geometric (×constant) | Equal ratios | 2, 6, 18, 54 |
| Square / Cube | Near n² or n³ values | 1, 4, 9, 16 |
| Square ± constant | n² off by fixed number | 3, 6, 11, 18 (n²+2) |
| Difference of differences | 2nd-level AP | 1, 3, 7, 13, 21 |
| Alternating (two series mixed) | Odd/even positions differ | 2, 10, 4, 20, 6, 30 |
| Prime series | | 2, 3, 5, 7, 11 |
| Multiplication with shift | ×n then ×(n+1) | 2, 4, 12, 48, 240 |
