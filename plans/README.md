# Study Plans

This folder holds actionable study plans, daily/weekly schedules, and progress logs.

## Suggested Structure

- `phase-1-foundation.md` — Initial phase covering basics across all subjects
- `phase-2-intensive.md` — Deep practice, mocks, and weak area focus
- `phase-3-revision.md` — Final revision before exams
- `progress-log.md` — Weekly/monthly progress entries
- Exam-specific plans as needed (e.g., `rbi-grade-b-plan.md`)
