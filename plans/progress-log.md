# Progress Log

One line per study session. Date in `YYYY-MM-DD` format. Keep entries terse — this is for Claude (and future-you) to resume context fast, not for reflection.

## Format

```
YYYY-MM-DD | <subject>/<chapter> | Stage <n> | <duration> | <status> | <note>
```

- **status**: `done` | `partial` | `stuck` | `revision`
- **note**: 1 line max. Only what's non-obvious (mistake pattern, insight, blocker). Skip "did X questions" — the day number already implies that.
