# Core Preparation Philosophy

Build one strong base that serves all three exams:

- **Conceptual depth** for RBI Grade B
- **Speed and pattern mastery** for SBI PO & IBPS PO

One engine, two driving modes: **analysis mode** and **turbo mode**.

## Guiding Principles

1. **Consistency over intensity** — No heroic 10-hour days. Show up daily. That's how ranks are quietly forged.
2. **Common base first, specialize later** — The 70-80% overlap is your foundation. Build it before branching.
3. **Practice > theory** — especially for Quant and Reasoning. Banking exams reward speed + accuracy, not just knowledge.
4. **Mock-driven feedback loops** — Start mocks early, even if scores are ugly. Treat analysis as sacred time.
5. **Quality of revision > quantity of new topics** — Multiple revisions of one source beats one read of many sources.
6. **One subject, one book, one current affairs source** — Anything extra must replace, not add.

## The Layered Approach

```text
Layer 1: Common Subject Base (70-80%)
  Quant + Reasoning + English + GA
  → Serves all three exams

Layer 2: RBI Core Layer
  Economics + Finance & Management + Descriptive
  → Your edge for RBI Grade B

Layer 3: PO Acceleration Layer
  Speed drills + timed sets + new patterns
  → Switch to turbo mode when PO exams approach
```

## Strategic Restraint — What to Postpone

- Computer awareness (easy to finish later, before exam)
- Static banking GK (after prelims notifications drop)
- Advanced descriptive answer polishing (Phase 2 time)

## Mental Model

Don't treat exams as separate mountains. Treat them as **different paths up the same mountain**.

- If RBI Grade B happens first, you're ready.
- If SBI/IBPS arrives earlier, you switch to turbo and sprint.

## Decision Checkpoint (at 3-4 months)

- If PO scores shoot up faster -> lean PO
- If descriptive answers start flowing -> lean RBI
- No doors slammed. Only doors waiting.

## Golden Rules for Working Professionals

- Low energy day? Read newspaper only. Still counts.
- Missed a day? Never compensate by overloading next day.
- Mocks feel bad? Good. They're diagnostics, not verdicts.
- Banking prep is repetitive by nature. The magic is not motivation, it's routine.
