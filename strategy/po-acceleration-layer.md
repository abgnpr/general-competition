# PO Acceleration Layer

Strategies specific to PO-level exams (IBPS PO, SBI PO) that layer on top of the common base.

## Speed Optimization

- Timed sets — practice under exam-like time pressure
- Mental math and shortcuts (Vedic math, approximation tricks)
- DI speed methods — quick calculation patterns
- RC scanning techniques

## Sectional Timing Strategy

- Prelims: speed + accuracy — pick questions wisely, skip tough ones early
- Mains: depth + analysis — manage time across more sections

## High-Frequency Topics

- Puzzles and seating arrangements are king (Reasoning)
- DI: tables, charts, caselets (Quant)
- RC is a daily non-negotiable (English)

## Prelims vs Mains Approach

- **Prelims:** Speed is everything. Pattern recognition. Quick wins first.
- **Mains:** Deeper analysis, data interpretation caselets, GA/Economy carry more weight.

## Mock Test Strategy (PO-specific)

- Sectional tests first (untimed, then timed)
- Progress to full-length mocks
- **Speed analysis** after every mock — where did time leak?
- If a mistake teaches you time management, it's PO gold.

## Descriptive Paper Prep

- Letter writing formats (formal/informal)
- Essay templates and structures
- Practice routine: start in Month 3-4
