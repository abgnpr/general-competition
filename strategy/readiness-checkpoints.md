## 3-4 Month Decision Checkpoint

After 3-4 months of common preparation, assess:

- If PO mock scores shoot up faster -> lean towards PO track
- If descriptive answers start flowing naturally -> lean towards RBI track
- No doors need to be closed. This is about **emphasis**, not elimination.

## Go / No-Go Criteria

Before appearing for an exam, check:

- [ ] Consistently scoring above target in mocks (last 3-5 mocks)
- [ ] All high-weight topics revised at least twice
- [ ] Last 3-4 months' current affairs revised 3-4 times
- [ ] Older months scanned at least 1-2 times
- [ ] Sectional timing is within limits
- [ ] Error rate on silly mistakes is trending down

## Reality Check

With the working professional plan (~3.5-4 hrs/weekday):

- **SBI / IBPS** -> Very realistic
- **RBI Grade B** -> Competitive if followed for 6-8 months
