# Weekly & Monthly Structure

## Weekly Subject Rotation

You will not study everything every day. Rotation keeps it manageable.

| Day | Morning (Core Subject) | Night Focus |
|-----|----------------------|-------------|
| Monday | Quant | Quant practice + CA |
| Tuesday | Reasoning | Reasoning practice + English |
| Wednesday | Quant | Quant practice + RBI Economics |
| Thursday | Reasoning | Reasoning practice + English |
| Friday | Quant | Quant practice + Finance |
| Saturday | Mixed + backlog | RBI Economics/Finance |
| Sunday | Revision | Mock/sectional + CA revision |

## Monthly Milestones

See [3-4-month-roadmap.md](../plans/3-4-month-roadmap.md) for the full month-by-month breakdown.

## Revision Weeks

- Schedule dedicated revision periods before each exam notification
- Last 2 weeks before an exam: no new topics, only revision + mocks

## Mock Test Cadence

| Phase | When | What |
|-------|------|------|
| Early | Month 1-2 | Sectional tests (untimed first, then timed) |
| Middle | Month 3 | Timed sectional tests (2-3 per week) + mixed practice sets |
| Late | Month 4+ | Full-length mocks (1 per weekend) |

### Dual Analysis Method

After every mock, analyze from two angles:

- **Speed analysis** (for SBI/IBPS): Where did time leak? Which questions took too long?
- **Concept gaps** (for RBI): What concept was unclear? What needs deeper understanding?

If a mistake teaches you a concept, it's **RBI gold**.
If it teaches you time management, it's **PO gold**.
